## 🚀 Quickstart (30 secondes)

```bash
# Terminal 1
docker-compose -f db/docker-compose.yml up -d

# Terminal 2
cd backend && ./mvnw spring-boot:run

# Terminal 3
cd frontend && npm install && npm run dev

# Browser
open http://localhost:5173
```
